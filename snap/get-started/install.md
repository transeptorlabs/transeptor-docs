This page explains how to install SmartHub Snap.

## Software Prerequisites
To install ERC-4337 Snap, you must install the [MetaMask Flask browser extension](https://metamask.io/flask/).


## Install the MetaMask Flask browser extension

[MetaMask Flask](https://metamask.io/flask/) is an experimental playground that provides developers 
access to upcoming MetaMask features. It is available as a browser extension.

:::caution Install in a new browser profile
Install the [Metamask Flask browser extension](https://chrome.google.com/webstore/detail/metamask-flask-developmen/ljfoeinjpaedjfecbmggjgodbgkmjkjk) in a new browser profile, or disable any existing installed versions of MetaMask before installing
Flask. Running multiple instances of MetaMask in the same browser profile breaks dapp interactions.
:::

## Install the SmartHub Snap

Follow steps to install the SmartHub Snap:

1. Click the link to access the SmartHub DApp. [https://transeptorlabs.github.io/smarthub-snap/](https://transeptorlabs.github.io/smarthub-snap/)
2. Once you're on the SmartHub DApp, follow these steps to install SmartHub Snap into your MetaMaskFlask:
   - Ensure that MetaMask Flask is installed and set up in your browser.
   - Locate the "Connect SmartHub Snap" button.
   - Click the button to initiate the installation process.
3. Follow any on-screen prompts from MetaMask to confirm the installation of SmartHub Snap.
4. Once the installation is complete, you're all set to explore and enjoy the features of SmartHub within the dapp.
5. Watch our demo for full walk-through of the installation process and how to use SmartHub Snap.
